# NEET 2024 Marks Distribution Analysis

This repository contains data, scripts, and figures for an analysis of the NEET 2024 marks distribution.

## Files

- `data.dat` — input marks/count data
- `fit.py` — fitting script
- `fit_parameters.dat` — fitted parameters
- `fit_components.dat` — fitted component data
- `fit_result.dat` — final fit result
- `fig1fig2.py` — script to generate Figs. 1 and 2
- `fig3.py` — script to generate Fig. 3
- `fig1.pdf`, `fig2.pdf`, `fig3.pdf` — generated figures

## How to run

python3 fit.py
python3 fig1fig2.py
python3 fig3.py
